# MarkDownViewer

MarkDownViewer is a lightweight, portable Markdown viewer for Windows. It renders GitHub Flavored Markdown with full support for tables, code blocks, task lists, Mermaid diagrams, and more.

## Download

Download the latest version from the [Releases page](../../releases).

### System Requirements
- **Windows 10** or later
- **WebView2 Runtime** — automatically installed on Windows 11; auto-installs on Windows 10 if needed

## Features

- ✅ **Full GFM Support** — GitHub Flavored Markdown rendering (tables, strikethrough, task lists, and more)
- ✅ **Syntax Highlighting** — Code blocks with language-specific highlighting
- ✅ **Mermaid Diagrams** — flowcharts, sequence diagrams, class diagrams, and more
- ✅ **Dark / Light Theme** — automatic system theme detection + manual toggle
- ✅ **Zoom** — adjustable text size (Ctrl+Plus, Ctrl+Minus)
- ✅ **Drag & Drop** — open files by dragging into the window
- ✅ **Tabbed Viewing** — open multiple files in one window
- ✅ **File Watching** — automatically reloads when files change on disk
- ✅ **Portable** — no installation required; just run the exe
