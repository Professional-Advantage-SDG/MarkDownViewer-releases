# How to Install MarkDownViewer

MarkDownViewer is a lightweight Markdown viewer built by Professional Advantage. This guide walks you through downloading and installing it for the first time.

## Step 1: Download

Click the download link (from Viva Engage, email, or the [Releases page](../../releases)) to download **MarkDownViewer-win-Setup.exe**.

## Step 2: Browser Download Warning

Your browser may show a warning like **"MarkDownViewer-win-Setup.exe isn't commonly downloaded"**. This is normal — Edge and Chrome flag any new software that hasn't been downloaded by millions of people yet.

**To proceed:**
1. Click the **⋯** (three dots) menu next to the download warning
2. Click **"Keep"**
3. If prompted again, click **"Show more"** → **"Keep anyway"**

> 💡 **Why does this happen?** Browsers use a reputation system based on download volume. Brand new software — even when properly signed — triggers this warning until enough people have downloaded it. The app is digitally signed by **Professional Advantage Pty Ltd**, confirming it's genuinely our software.

## Step 3: Run the Installer

Double-click **MarkDownViewer-win-Setup.exe** to start the installation.

## Step 4: Windows SmartScreen

You may see a blue **"Windows protected your PC"** screen. This is another reputation check — it happens once for any new software.

**To proceed:**
1. Click **"More info"** (small text link below the warning message)
2. Verify the publisher shows **"Professional Advantage Pty Ltd"** — this confirms the app is signed by us
3. Click **"Run anyway"**

> 💡 **This only happens once.** After installation, you won't see this warning again. Future updates are installed automatically in the background.

## Step 5: You're Done! 🎉

MarkDownViewer will install and open automatically. You can now:
- **Open any .md file** by double-clicking it
- **Drag and drop** Markdown files into the window
- Use **File → Open** to browse for files

Future updates are handled automatically — you'll always have the latest version without needing to download anything again.

## Alternative: Unblock via File Properties

If you prefer, you can remove the security flag before running the installer:

1. Right-click **MarkDownViewer-win-Setup.exe** in your Downloads folder
2. Click **Properties**
3. At the bottom of the General tab, check the **"Unblock"** checkbox
4. Click **OK**
5. Now double-click the installer — SmartScreen won't appear
